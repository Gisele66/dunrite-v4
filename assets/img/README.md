# Images folder

## Folder layout

```
assets/img/
├── logo.svg              ← Dark logo (sticky header, light backgrounds)
├── logo-light.svg        ← Light logo (hero / dark backgrounds)
├── favicon.svg           ← Browser tab icon
└── photos/               ← **Drop your photos here**
    ├── README-PHOTOS.md  ← Filename checklist
    ├── dave.jpg
    ├── dave-michelle.jpg
    ├── melina.jpg
    ├── washing.jpg
    ├── pickup.jpg
    ├── fleet-1.jpg
    ├── fleet-2.jpg
    └── crew.jpg          ← optional
```

## Quick start

1. Open **`photos/`** in File Explorer.
2. Copy your image files in using the **exact filenames** in `photos/README-PHOTOS.md`.
3. Refresh the browser — images appear automatically (no code changes).

**Format:** `.jpg` recommended · **Width:** 1600–2400px is enough · **Optimize** before upload if files are very large.

If a file is missing, the site shows a branded placeholder until you add it.
